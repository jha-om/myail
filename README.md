# Myail
<h3>An alternative to Gmail but with AI integration.</h3>

# Tech Stack

- <b>Frontend:</b> Next.js, React, TypeScript, TailwindCSS, Shadcn UI  
- <b>Backend:</b> Node.js, Prisma  
- <b>Database:</b> PostgreSQL  
- <b>Authentication:</b> Clerk
- <b>Email API:</b> Aurinko
